# EventSync

**EventSync** is a web app that helps students and organizers manage and participate in events efficiently.

## Features
- Student & Organizer dashboards
- Event creation, registration, and tracking
- Responsive UI with reusable components
- Uses local storage/hooks for data

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Vite
- Git & GitHub

## Installation
```bash
git clone https://github.com/madhurima7284/EventSync.git
cd EventSync
npm install
npm run dev
